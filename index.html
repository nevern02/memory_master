Memory master

An HTML5 game in the style of Concentration.

Download this project as a .zip file Download this project as a tar.gz file

Memory Master is a card matching game in the style of the classic Concentration. It's an HTML5 game built using Angular and standard CSS animations. The game is free to play, but willing supporters may optionally purchase it from the Chrome App Store.
